# genpost

genpost 是markdown文件生成器

## 安装方法

```bash
git clone https://github.com/hidevopsio/genpost
cd genpost
go install
```

## 生成文章类别

在项目文件夹下面运行 `genpost -c` 按照提示输入文章类别的目录和标题即可

```bash

genpost -c

✔ 目录: articles
✔ 标题: 分享文章
✔ 排序: 1
```

## 生成文章

在项目文件夹下面运行 `genpost` 按照提示选择文章类型，输入文章标题和作者即可
